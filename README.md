# Jogo da Menina Escapando do Lixo

Este é um jogo simples onde uma menina precisa escapar de um cenário de lixo, coletando o lixo e jogando-o nos recipientes corretos.

## Objetivo do Jogo

O objetivo do jogo é controlar a personagem (uma menina) para pegar o lixo que cai do topo da tela e jogá-lo nos recipientes certos. A cada colisão bem-sucedida, o lixo é descartado e um novo lixo começa a cair.

## Como Jogar

- Use as teclas **seta para a esquerda** e **seta para a direita** para mover a menina.
- A menina deve pegar o lixo e jogá-lo nos recipientes corretos (três recipientes diferentes na parte inferior da tela).
- O jogo termina se você errar ou o lixo não for jogado nos recipientes corretos.

## Tecnologias Usadas

- **HTML** para a estrutura da página.
- **CSS** para o design do jogo.
- **JavaScript** para a lógica do jogo.

## Como Executar o Jogo

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Comece a jogar!
